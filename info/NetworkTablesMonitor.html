What is NetworkTablesMonitor?

NetworkTablesMonitor is a tool that is mainly used for viewing NetworkTables data, NetworkTables is a api developed by FIRST©.
NetworkTablesMonitor can also be used for reading and writing to tcp sockets.

NetworkTablesMonitor can act as a NetworkTables server or connect to a server using an ip address or a team number and can publish strings, integers and booleans as well as lists of those types to a NetworkTables server.
NetworkTablesMonitor can also listen on tcp ports and connect to a server using an ip address and port and it can read and write strings to a currently open socket.

Planned features
